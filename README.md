# ml-multimodal-skin-cancer-study

Repositório para armazenar o material de estudo relacionado à pesquisa sobre detecção multimodal de câncer de pele.

## Conteúdo

- `Deep_Learning_Multimodal_para_Detecção_de Câncer_de_Pele.pdf`

## Descrição

Este estudo pretende investigar se a integração entre imagem clínica dermatológica e uma descrição clínica curta em português do Brasil pode melhorar a triagem automatizada de lesões pigmentadas suspeitas de melanoma, em comparação com abordagens unimodais.

O foco é avaliar se a incorporação de informação textual estruturada, associada à análise visual por modelos de inteligência artificial, contribui para classificações mais precisas, robustas e contextualizadas.

## Objetivo do Repositório

Este repositório funciona como base documental para o estudo e para a organização do material de apoio relacionado à linha de pesquisa.
